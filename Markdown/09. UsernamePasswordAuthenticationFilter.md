# UsernamePasswordAuthenticationFilter

> 폼 로그인을 처리하는 인증 필터

- 사용자가 폼에 입력한 username 과 password 로 Authentication 을 만들고 AuthenticationManager 를 사용하여 인증을 시도한다.
- AuthenticationManager(ProviderManager) 는 여러 AuthenticationProvider 를 사용하여 인증을 시도하는데, 그 중에 DaoAuthenticationProvider 는
UserDetailsService 를 사용하여 UserDetails 정보를 가져와 사용자가 입력한 password 와 비교한다.

로그인 페이지에서 username 과 password 를 입력하면 UsernamePasswordAuthenticationFilter 를 통해 처리가 된다.
